| Status | Meaning |
|---|---|
| `idea` | Proposed, not yet discussed |
| `draft` | Under active design |
| `accepted` | Design agreed; ready for implementation |
| `implemented` | Exists in the compiler |
| `deferred` | Intentionally postponed |
| `rejected` | Will not be included (with documented rationale) |
